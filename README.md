# Belajar Bahasa Rust

### Commands
```bash
########### FILE ###########
# compile file
rustc file_name.rs

# compile file with option output executable file name
rustc -o output_exe file_name.rs

# run file
./file_name

# or 
./output_exe

########### PROJECT ###########
# creat project
cargo new project_name

# build project
cargo build

#run project
cargo run

# run the binaru file
./target/debug/project_name_binary
```
